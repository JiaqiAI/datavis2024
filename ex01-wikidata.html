Exercice no. 01

Ligne de temps faite avec Wikidata

Cette timeline montre des universities dans different pays.

La requête est :


                #defaultView:Timeline
                SELECT ?university ?universityLabel ?inception ?image ?country ?countryLabel ?students_count ?coordinate_location WHERE {
                SERVICE wikibase:label { bd:serviceParam wikibase:language "[AUTO_LANGUAGE],en". }
                ?university wdt:P31 wd:Q3918.
                OPTIONAL { ?university wdt:P571 ?inception. }
                OPTIONAL { ?university wdt:P18 ?image. }
                OPTIONAL { ?university wdt:P17 ?country. }
                OPTIONAL { ?university wdt:P2196 ?students_count. }
                OPTIONAL { ?university wdt:P625 ?coordinate_location. }
                }
                LIMIT 100
